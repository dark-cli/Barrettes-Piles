# Nomenclature

## Symbols

### Geometric Parameters
- $B$ = Width of barrette (m)
- $L$ = Length of barrette (m), typically $L > B$
- $D$ = Embedment depth (m)
- $A_g$ = Gross cross-sectional area (m²)
- $A_s$ = Reinforcement area (m²)
- $I$ = Moment of inertia (m⁴)
- $d$ = Effective depth (m)

### Material Properties
- $f_c'$ = Concrete compressive strength (MPa)
- $f_y$ = Reinforcement yield strength (MPa)
- $E$ = Modulus of elasticity (kN/m²)
- $\nu$ = Poisson's ratio
- $c'$ = Effective cohesion (kN/m²)
- $c_u$ = Undrained shear strength (kN/m²)
- $\phi'$ = Effective friction angle (degrees)
- $\gamma$ = Unit weight of soil (kN/m³)
- $E_s$ = Soil modulus (kN/m²)

### Loads
- $P$ = Axial load (kN)
- $Q$ = Total load (kN)
- $M$ = Moment (kN·m)
- $V$ = Shear force (kN)
- $H$ = Lateral load (kN)
- $q$ = Bearing pressure (kN/m²)
- $p$ = Soil reaction per unit length (kN/m)

### Capacities
- $Q_u$ = Ultimate capacity (kN)
- $P_u$ = Ultimate axial capacity (kN)
- $M_u$ = Ultimate moment capacity (kN·m)
- $V_u$ = Ultimate shear capacity (kN)
- $q_u$ = Ultimate bearing capacity (kN/m²)

### Settlement and Displacement
- $s$ = Settlement (m)
- $y$ = Lateral deflection (m)
- $\delta$ = Displacement (m)

### Factors
- $N_c$, $N_q$, $N_\gamma$ = Bearing capacity factors
- $s_c$, $s_q$, $s_\gamma$ = Shape factors
- $d_c$, $d_q$, $d_\gamma$ = Depth factors
- $\alpha$ = Adhesion factor
- $\beta$ = Skin friction factor
- $\phi$ = Strength reduction factor (concrete design)
- $\gamma_R$ = Resistance factor (geotechnical design)

## Subscripts
- $b$ = Base
- $s$ = Shaft
- $conc$ = Concrete
- $strong$ = Strong axis
- $weak$ = Weak axis
- $design$ = Design value
- $u$ = Ultimate
- $c$ = Characteristic

## Greek Letters
- $\alpha$ = Alpha - adhesion factor, angle
- $\beta$ = Beta - skin friction factor, angle
- $\gamma$ = Gamma - unit weight, partial factor
- $\delta$ = Delta - interface friction angle, deflection
- $\epsilon$ = Epsilon - strain
- $\phi$ = Phi - friction angle, strength reduction factor
- $\rho$ = Rho - reinforcement ratio
- $\sigma$ = Sigma - stress
- $\tau$ = Tau - shear stress
- $\psi$ = Psi - dilation angle, modification factor

## Abbreviations
- FEA = Finite Element Analysis
- FEM = Finite Element Method
- FHWA = Federal Highway Administration
- ICE = Institution of Civil Engineers
- FOS = Factor of Safety
- SPT = Standard Penetration Test
- CPT = Cone Penetration Test
- SSI = Soil-Structure Interaction
- LRFD = Load and Resistance Factor Design
