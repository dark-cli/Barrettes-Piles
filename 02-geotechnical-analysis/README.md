# Geotechnical Analysis

## Overview

Geotechnical analysis of barrettes encompasses bearing capacity evaluation, settlement prediction, lateral resistance, and soil-structure interaction. The rectangular cross-section of barrettes requires modifications to traditional pile analysis methods.
